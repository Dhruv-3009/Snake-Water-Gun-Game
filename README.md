# Snake Water Gun Game 🐍💧🔫

## Description

This is a simple Python game based on the Snake-Water-Gun concept. The user plays against the computer, and the winner is decided based on predefined rules.

## Features

* User vs Computer gameplay
* Random computer choice
* Simple and beginner-friendly

## How to Run

1. Install Python
2. Run the file:
   python game.py

## Technologies Used

* Python
